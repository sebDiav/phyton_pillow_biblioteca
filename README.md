# Biblioteca Pillow en Python

## 📌 Descripción

Este proyecto presenta el uso de la biblioteca Pillow en Python para el procesamiento básico de imágenes.

Se realiza un ejemplo práctico que permite:

- Abrir una imagen.
- Mostrar sus propiedades (tamaño, formato y modo).
- Redimensionarla.
- Convertirla a escala de grises.
- Guardar la imagen modificada.

Este trabajo fue desarrollado como parte de una actividad académica sobre el uso de bibliotecas en Python.

---

## 📚 Biblioteca utilizada

**Pillow** es una biblioteca de Python que permite abrir, manipular y guardar imágenes en diferentes formatos como JPG, PNG, BMP, GIF, entre otros.

Es ampliamente utilizada en procesamiento digital de imágenes y desarrollo de aplicaciones gráficas.
